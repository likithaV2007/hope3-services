# Hope3 Services

A comprehensive Flutter application suite consisting of multiple modules for ride-sharing and transportation services.

## Project Structure

This repository contains the following modules:

- **authentication_module** - User authentication and authorization
- **driver_app** - Driver-facing mobile application
- **fcm_notification_module** - Firebase Cloud Messaging notifications
- **location_module** - Location tracking and services
- **network_checking_module** - Network connectivity monitoring
- **parent_app** - Main parent/passenger application

## Getting Started

Each module is a separate Flutter project with its own dependencies and configuration.

### Prerequisites

- Flutter SDK (latest stable version)
- Dart SDK
- Android Studio / VS Code
- Firebase account (for FCM and authentication)

### Installation

1. Clone this repository
2. Navigate to each module directory
3. Run `flutter pub get` in each module
4. Configure Firebase for modules that require it
5. Run the applications using `flutter run`

## Development

Each module can be developed and tested independently. Refer to individual module README files for specific setup instructions.
